# Prework Study Guide Webpage

## Description

This project is created to help the students summarize the material they learned in Prework. It includes information about HTML, CSS, Git, and Javascript.



## Installation

N/A

## Usage

The students can go back and review their notes by accessing this project. There are 4 sections, and each of them contains information about HTML, CSS, Git, or Javascript. We can see the Javascipt running by opening the DevTools console. It will also give us suggestion on what topic to study first.
